Comment l'installer ?

Prêt à fonctionner sur ton serveur en moins de 2 minutes chrono.

01

Invitation

Clique sur le bouton "Ajouter à Discord" et sélectionne le serveur de ton choix parmi ta liste.

02

Autorisations

Accorde les permissions demandées (Modération, Lecture/Écriture) pour que le bot fonctionne au maximum de ses capacités.

03

Configuration

Utilise la commande /setup directement sur ton serveur pour lancer l'assistant de configuration automatique.

Les Commandes Clefs

Une interface moderne utilisant exclusivement les dernières Slash Commands de Discord.

Commande Description Permission requise
/help Affiche la liste complète de toutes les commandes classées par catégories. Aucune
/ban [membre] [raison] Bannit définitivement un utilisateur fautif du serveur et lui envoie un MP. Aucune
/assistance Demande une assistance pour ton serveur en cas de raid Aucune
/config Configure les fonctionnalitées comme l' antiraid, l' antilien, l' antispam, ... Aucune
